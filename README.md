# cyart-vapt-week-1
This repository presents my VAPT internship project completed at CyArt Tech. It demonstrates hands-on experience in vulnerability assessment, network scanning, risk analysis, and professional reporting using tools like Kali Linux, Nmap, OpenVAS, and Metasploit in an authorized lab environment.

Overview
This repository contains my hands-on Vulnerability Assessment and Penetration Testing (VAPT) project completed during my internship at CyArt Tech. The project demonstrates practical implementation of security assessment techniques in a controlled lab environment.

Objective
The objective was to identify, analyze, and validate vulnerabilities while following industry-standard VAPT methodology and professional reporting practices.

Tools Used
Kali Linux
Nmap
OpenVAS (Greenbone)
Nikto
Metasploit Framework
OWASP ZAP
Dradis CE

Lab Environment
Attacker Machine: Kali Linux
Target Machine: Metasploitable
Network Setup: VirtualBox (Internal Network)
Target IP: 192.168.207.129

All testing was conducted in an authorized and isolated virtual lab for educational purposes only.

Methodology Followed
The project was executed using a structured VAPT approach:
Planning – Defined scope, target IP, and authorization
Discovery – Network scanning and service enumeration
Vulnerability Assessment – Identified weaknesses using OpenVAS and Nikto
Exploitation (Controlled) – Validated vulnerabilities using Metasploit
Risk Assessment – Applied CVSS scoring and risk matrix
Reporting – Documented findings with evidence and remediation steps

Key Vulnerabilities Identified
FTP Backdoor Command Execution (Critical)
Samba Remote Code Execution (Critical)
Apache Path Traversal (High)
Weak MySQL Authentication (Medium)
Risk prioritization was performed using CVSS scoring and impact analysis.

Key Learning Outcomes
Practical experience with industry security tools
Understanding of real-world vulnerability analysis
Application of CVSS scoring methodology
Professional security documentation and reporting
Exposure to structured cybersecurity workflow
